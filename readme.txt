hhu-esag-theater
================

Skript zum ESAG Theater

Anleitung:

- jeder Sketch in eine eigene Datei, speichern unter sketche/sketchname.tex
- in der theater.tex lediglich "\include{sketche/sketchname}" an entsprechender Stelle einfügen
- wenn \newcommand benutzt wird, bitte in theater.tex oben einfügen
